Features

Built for jewelers, by engineers

From catalog management to immutable provenance tracking — everything a luxury atelier needs.

💎

Jewel Catalog

CRUD with full validation, QR codes, AI-powered descriptions via local LLM (Ollama), and certificate generation.

🔗

Hashchain

SHA256-linked immutable audit trail per jewel. Every operation creates a tamper-evident entry in the chain.

💰

Financial Ledger

Double-entry accounting with automated transaction creation on sale. Channel-based profit tracking and real-time P&L.

🖥️

Web Terminal

Real browser-based terminal via xterm.js + WebSocket. Multiple tabs, tmux persistence, Ctrl+Shift+T shortcut.

📊

Monitoring

Unified health endpoint checking 5 services in parallel. Service status dashboard with live updates.

🔄

Blue-Green Deploy

Atomic Nginx switch with health checks and instant rollback. Zero-downtime deployments with systemd.

Quick Start

Up and running in 3 steps

Terminal 
# Clone and install
git clone https://github.com/Sebastiano-cr/vivify-luxury-platform.git
cd vivify-luxury-platform
pip install -r requirements.txt

# Start backend
python -m uvicorn backend.server:app --port 3334

# Open dashboard
http://localhost:3334
